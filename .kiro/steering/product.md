# Product Overview

**Codementee** is a mentorship-based interview preparation platform that connects aspiring software engineers with experienced mentors from top product-based companies for mock interviews.

## Core Value Proposition
"Real mock interviews with engineers who've cracked product-based companies like Amazon, Google, Microsoft, and more."

## Key User Roles
- **Admin**: Platform management, user oversight, revenue tracking, system configuration
- **Mentor**: Conducts mock interviews, provides structured feedback, confirms booking requests
- **Mentee**: Books mock interviews, receives feedback, practices with company-specific preparation

## Business Model
- Subscription-based plans: Monthly (₹1,999), Quarterly (₹4,999), Biannual (₹8,999)
- Payment processing via Razorpay (live integration)
- Automated email notifications via Resend

## Core Features
- **Booking System**: 3-step wizard for mentees to request mock interviews
- **Auto-assignment**: Google Meet links from managed pool
- **Feedback System**: Structured post-interview evaluations
- **Role-based Dashboards**: Tailored interfaces for each user type
- **Payment Integration**: Seamless onboarding with automatic account creation

## Target Companies
Focus on preparation for interviews at major product-based companies including Amazon, Google, Microsoft, and other top-tier tech companies.
